# Customer Intake Rules

The Free Preview form should collect enough detail for a useful draft without turning into a long consultation.

## Important Fields

- Name
- Business name
- Website
- Email
- Phone
- Industry
- Main services
- Main lead sources
- Biggest lead bottleneck
- Current tools/CRM
- What happens after a lead comes in
- Anything else SignalOps should know

## Classification Rules

- One lead source and simple needs usually points to Starter.
- Multiple lead sources, quote follow-up, photo/detail collection, dashboard needs, or booking handoff usually points to Growth.
- Complex routing, multiple locations, CRM integration, custom agent needs, or custom dashboards point to Custom.
- Emergency/urgent language routes toward Emergency Response OS.
- Quote/photo/service estimate language routes toward Quote Intake OS.
- Consultation/appointment language routes toward Appointment Booking OS.
- Insurance, real estate, law, and longer sales cycles route toward Lead Nurture OS.

## Missing Info Rules

The Prompt Worker should flag missing information rather than inventing details. Common missing fields:

- Website URL
- Best phone number
- Main services
- Lead sources
- Current tools/CRM
- Current lead process
- Average job value
- Monthly lead volume
