# System Templates

## Quote Intake OS

For mobile fleet wash companies, detailers, contractors, tint/wrap shops, remodelers, auto services, and quote-based businesses.

Core flow: lead comes in -> AI requests missing details, photos, or account context -> sorts priority and next action -> alerts owner -> follow-up -> booking/quote handoff.

## Appointment Booking OS

For med spas, dental, wellness, clinics, and service appointments.

Core flow: lead asks about service -> AI collects the right details -> booking handoff -> no-book follow-up.

## Emergency Response OS

For HVAC, plumbing, roofing, restoration, well/water, and urgent service businesses.

Core flow: urgent inquiry -> AI collects issue/address/urgency -> hot alert -> callback handoff -> follow-up.

## Lead Nurture OS

For insurance, real estate, B2B services, and high-consideration sales.

Core flow: lead captured -> need/timeline collected -> tagged hot/warm/cold -> nurture sequence -> sales next action.

## Custom Ops OS

For complex workflows, multi-location businesses, CRM-heavy teams, advanced routing, custom dashboards, or custom internal AI systems.

Core flow: request enters system -> AI classifies workflow path -> routing/approval/data rules run -> team gets next action -> dashboard and follow-up stay updated.
