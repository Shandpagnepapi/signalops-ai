# Implementation Checklist

## Setup

- Confirm primary lead sources
- Confirm CRM and calendar tools
- Confirm notification channels
- Create environment variables
- Configure staging and production URLs

## Lead Intake

- Map form fields
- Validate required fields
- Normalize phone and email data
- Add source tracking
- Create lead scoring rules

## Qualification

- Define high-intent keywords
- Define lead value signals
- Define disqualification criteria
- Create recommended next actions
- Test sample leads across A, B, and C grades

## Follow-Up

- Draft first-response templates
- Draft nurture replies
- Draft missed-call recovery messages
- Configure owner/team alerts
- Prepare calendar routing rules

## Dashboard

- Track new leads
- Track median first response time
- Track qualified lead volume
- Track booked appointments
- Track follow-up completion rate
