# SignalOps Pilot Offer

## Pilot Positioning

This is not a full digital transformation project.

This is a focused pilot to prove one thing quickly: better lead response and follow-up should produce more booked appointments from existing lead flow.

## Pricing

- Pilot setup: $1,500 to $3,000
- Pilot management: $500/month
- Optional add-ons: advanced integrations, custom AI workflows, multi-location routing

## Deliverables

- Missed lead check and implementation map
- One primary lead-source workflow (form/call/ad/DM)
- Instant response logic with intake prompts
- Priority sorting and routing rules
- Owner/sales alert workflow
- Follow-up sequence starter (Day 1, Day 2, Day 5, Day 10)
- Dashboard visibility for pilot leads

## Timeline

- Week 1: Audit, workflow mapping, tool access, implementation plan
- Week 2: Build and configure response, intake, and routing
- Week 3: Launch, monitor, and tune
- Week 4: Performance review and expansion recommendation

## Expectations

Client responsibilities:
- Provide access to current tools (forms, CRM, calendar, inboxes)
- Approve messaging and routing rules
- Assign a point of contact for decisions
- Respond quickly to testing feedback

SignalOps responsibilities:
- Build practical workflows tied to booking outcomes
- Keep implementation scope clear and measured
- Report findings and recommendations transparently

## Testimonial and Case Study Ask

If pilot goals are met, we ask for:
- Short testimonial
- Permission for a case study summary
- Permission to reference results in sales conversations

No confidential data is shared without explicit approval.

## Scope Boundaries

Included:
- Lead response and intake workflows
- Basic notification and follow-up logic
- Core dashboard/pipeline visibility

Not included in pilot by default:
- Full CRM replacement
- Full outbound sales system
- Complex multi-branch operational redesign
- Paid media management
- Custom software beyond agreed pilot scope

## Automotive Service Businesses Version

This pilot is ideal for auto shops, wheel repair shops, detailers, tint/wrap shops, and performance shops.

Automotive-specific pilot outcomes:
- Faster quote response for repair/refinish requests
- Structured photo intake for cleaner estimates
- Urgent/safety-sensitive flagging (bent, cracked, non-drivable)
- Smart routing:
  - Mobile cosmetic repair opportunities
  - In-shop inspection opportunities
- Follow-up automation until booking

Automotive proof metric examples:
- Speed-to-lead improvement
- Higher quote-to-book rate
- Fewer dropped quote requests
- Better high-value lead prioritization
